# AA6_ENIGMA_Diana_Roc
Simulación básica de la máquina ENIGMA desarrollada en C++ usando la biblioteca estándar. Implementa un sistema de cifrado y descifrado reversible con tres rotors configurables mediante ficheros. El programa procesa texto en mayúsculas, elimina caracteres no válidos y ofrece un menú de consola para cifrar, descifrar y editar rotors.
